1° Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags , e e a instrução . Esta estrutura está informada em uma documentação que descreve todos os detalhes do HTML, no caso as tags e atributos, e como os navegadores devem considerar e interpretar estas tags, esta documentação é chamada de "especificação do HTML", e através do que está declarado nela que é possível entender se um documento HTML válido.

2° Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros

(Voltar)

3° O que são os atributos em HTML? Resultado de imagem para o que é atributo html Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc…

(Voltar)

4° As tags padrão para títulos que já estão pré-formatadas nos navegadores Web são: h1, h2, h3, h4, h5 e h6. Estas tags já estão formatadas em negrito e com uma letra maior do que a do texto convencional. A única diferença entre estas tags é que a h1 possui a maior tamanho de letra e a letra vai diminuindo de tamanho até chegar até a h6, que possui o menor tamanho de letra.

(Voltar)

5° O Elemento HTML ol (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.

(Voltar)

6° Para criar um link em uma página da web, você precisa envolver um elemento (um texto, uma imagem e assim por diante) em um elemento de âncora ( a ) e definir seu atributo href para que aponte para o URL o qual você quer associar

  • Desenvolvimento web. Os desenvolvedores usam códigos HTML para projetar como um navegador vai exibir os elementos das páginas, como textos, hiperlinks e arquivos de mídia.
  • Navegação na internet. Os usuários podem navegar facilmente e inserir links entre páginas e sites relacionados, já que o HTML é amplamente usado para incorporar hiperlinks.
  • Documentação. O HTML torna possível a organização e a formatação de documentos, de maneira similar ao Microsoft Word.

Também vale notar que o HTML agora é considerado um padrão oficial da internet. O World Wide Web Consortium (W3C) mantêm e desenvolve especificações do HTML, além de providenciar atualizações regulares.

Este artigo vai cobrir o básico sobre HTML, incluindo como ele funciona, seus prós e contras — além de como ele se relaciona com CSS e JavaScript.
